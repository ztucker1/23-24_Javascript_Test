Javascript Unit Test

Basic Directions

This project is meant to assess your understanding of Javascript usage within a webpage. There are five tasks that you must complete. The tasks are functionality that will be added to this page. Inside of the folder you found this html file, you will see a folder called "assets". Within that, another folder called "javascript" has 5 files labeled task1.js to task5.js. For each task, place your javascript code within the appropriate file. The files are already linked to this page in the head of this document. The instructions for each task are listed below.

Tasks

The tasks that you need to accomplish are listed below. If you have any questions, feel free to ask me. If it is over the weekend, feel free to email me at akiedes@vtsd.com. If there is a place on the webpage where an element needs to be added for a task, place it within the area labeled for that task.

  1. When you hover the mouse over the image, a button should appear. When you click the button, the image should disappear.
  2. Create 3 buttons (similar to the one provided for "Task 1") each associated with a different color. When each button is clicked, change the background of the table to the color the button represents.
  3. Add a button to the navigation bar which, when clicked changes the text within the basic directions to "I Clicked It!".
  4. Create a button within the navigation called "clear" which opens a new page with the text "new page has been created".
  5. On the form add validation functionality that shows the form field (or its label) as red if the field is empty. They should turn back to white when the data is entered.
